# psgg-cpphppwfuncptr-sample
C++ using Function Pointer,cpp and hpp file sample.
Generate a declaration part (hpp) and implementation part (cpp) file.

## ENVIRONMENT

Windows 10/11  
Visual Studio 2022
[StateGo](https://statego.programanic.com/index-e.html)
